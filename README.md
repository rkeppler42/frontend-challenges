## 🇧🇷 Português

Repositório central de desafios frontend resolvidos em múltiplas plataformas.  
O objetivo é praticar HTML, CSS, JavaScript e bibliotecas/frameworks como exercícios de lógica e percepção visual — complementando minha formação como desenvolvedor **full-stack**.
