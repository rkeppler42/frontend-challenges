## 🇺🇸 English

A central repository for frontend challenges solved across multiple platforms, including Frontend Mentor, GreatFrontEnd, roadmap.sh, WebDevDaily, DevChallenges, BigDevSoon, and iCodeThis.

The goal is to practice HTML, CSS, JavaScript, responsive design, accessibility, and UI implementation as logic and visual puzzles — complementing my path toward becoming a **full-stack developer focused on Python**.

### 🧠 Concepts Practiced

- Semantic HTML
- CSS layouts
- Flexbox and Grid
- Responsive design
- Accessibility basics
- JavaScript interactions
- UI implementation from visual references
- Component-based thinking
- Clean project organization
- Documentation and deployment workflow

### 🌐 Platforms

| Platform | Focus |
| --- | --- |
| [Frontend Mentor](https://www.frontendmentor.io/) | HTML, CSS, responsive layouts |
| [GreatFrontEnd](https://www.greatfrontend.com/) | Frontend engineering, JavaScript, interviews |
| [roadmap.sh](https://roadmap.sh/) | Guided projects and skill paths |
| [WebDevDaily](https://webdevdaily.io/) | Daily UI practice |
| [DevChallenges](https://devchallenges.io/) | App-like projects |
| [BigDevSoon](https://bigdevsoon.me/) | Visual UI components |
| [iCodeThis](https://icodethis.com/) | Daily UI puzzles |

### 📁 Repository Structure

```txt
frontend-challenges/
├── frontendmentor/
├── greatfrontend/
├── roadmap-sh/
├── webdevdaily/
├── devchallenges/
├── bigdevsoon/
└── icodethis/
```

Each platform folder has its own `README.md` with the list of completed challenges.

### 🚀 How to View

Most challenges can be opened directly in the browser.

For simple HTML/CSS/JavaScript projects:

```bash
# Open the challenge folder
# Then open index.html in your browser
```

Some challenges may also include a live demo link when deployed.

## 🇧🇷 Português

Repositório central de desafios de frontend resolvidos em múltiplas plataformas, incluindo Frontend Mentor, GreatFrontEnd, roadmap.sh, WebDevDaily, DevChallenges, BigDevSoon e iCodeThis.

O objetivo é praticar HTML, CSS, JavaScript, design responsivo, acessibilidade e implementação de interfaces como exercícios de lógica e percepção visual — complementando minha formação como desenvolvedor **full-stack com foco em Python**.

### 🧠 Conceitos Praticados

- HTML semântico
- Layouts com CSS
- Flexbox e Grid
- Design responsivo
- Noções de acessibilidade
- Interações com JavaScript
- Implementação de UI a partir de referências visuais
- Pensamento baseado em componentes
- Organização limpa de projetos
- Documentação e fluxo de deploy
