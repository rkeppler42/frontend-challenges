# The Odin Project — Odin Recipes

## Overview

A solution to the Odin Recipes project from The Odin Project.  
This project focuses on basic semantic HTML, page structure, links, lists, images, and building a small multi-page website without CSS or JavaScript.

### Links

- **Live Demo:** [rkeppler42.github.io/frontend-challenges/the-odin-project/01-recipes](https://rkeppler42.github.io/frontend-challenges/the-odin-project/01-recipes/)
- **Project:** [theodinproject.com/lessons/foundations-recipes](https://www.theodinproject.com/lessons/foundations-recipes)
- **Code:** [github.com/rkeppler42/frontend-challenges/tree/main/the-odin-project/01-recipes](https://github.com/rkeppler42/frontend-challenges/tree/main/the-odin-project/01-recipes)

## Built with

- Semantic HTML5
- Multi-page HTML structure
- Lists
- Links
- Images
- Basic file organization

## What I learned

I practiced creating a basic multi-page website using only HTML. This project helped reinforce HTML boilerplate structure, relative links, unordered and ordered lists, image embedding, and organizing recipe pages inside a dedicated folder.
