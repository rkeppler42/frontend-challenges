# The Odin Project — Landing Page

## Overview

A solution to the Landing Page project from The Odin Project.  
This project focuses on recreating a provided design from scratch using semantic HTML, CSS, Flexbox, external stylesheets, and basic visual layout techniques.

- **Live Demo:** [rkeppler42.github.io/frontend-challenges/the-odin-project/02-landing-page](https://rkeppler42.github.io/frontend-challenges/the-odin-project/02-landing-page/)

## Built with

- Semantic HTML5
- CSS
- CSS Flexbox
- CSS custom properties
- External stylesheet
- Google Fonts
- Desktop layout workflow

## What I learned

I practiced building a full landing page from a visual reference using HTML and CSS. This project helped reinforce Flexbox layout, page sectioning, spacing, alignment, typography, color organization, and translating a static design into a structured web page.
