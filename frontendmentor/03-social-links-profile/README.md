A solution to the Social Links Profile challenge from Frontend Mentor.  
This project focuses on semantic HTML, responsive layout, CSS custom properties, and clean component styling.

## Built with

- Semantic HTML5
- CSS Flexbox
- CSS custom properties
- Variable font with `@font-face`
- Mobile-first workflow

## What I learned

I practiced organizing CSS custom properties for colors, typography presets, and spacing tokens. I also learned that fixed dimensions should come from padding and layout calculations rather than arbitrary values.
